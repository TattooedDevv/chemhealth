# User Stories

## Researcher Stories

US1:
As a researcher,
I want to upload molecular datasets,
So that I can analyze chemical properties.

US2:
As a researcher,
I want to visualize molecular structures,
So that I can understand bonding patterns.

US3:
As a researcher,
I want to run machine learning models,
So that I can predict absorption rates.

US4:
As a researcher,
I want to download analysis reports,
So that I can use them for further study.

## Student Stories

US5:
As a student,
I want to explore chemical data,
So that I can learn computational chemistry.

US6:
As a student,
I want to compare multiple compounds,
So that I can understand differences in behavior.

US7:
As a student,
I want to view interactive charts,
So that I can interpret results easily.

## Administrator Stories

US8:
As an administrator,
I want to manage datasets,
So that system data remains accurate.

US9:
As an administrator,
I want to monitor system performance,
So that reliability is maintained.

US10:
As a researcher,
I want to retrieve external chemical data,
So that I can enrich my analysis.
